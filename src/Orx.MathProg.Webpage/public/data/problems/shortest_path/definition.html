Shortest Path Problem

Wikipedia

In graph theory, the shortest path problem is the problem of finding a path between two vertices (or nodes) in a graph such that the sum of the weights of its constituent edges is minimized.

The problem of finding the shortest path between two intersections on a road map may be modeled as a special case of the shortest path problem in graphs, where the vertices correspond to intersections and the edges correspond to road segments, each weighted by the length of the segment.

Mathematical Formulation

Let `N` be set of nodes and `(i,j) \in A` be an arc of the graph `G = (N, A)` with weight `w_{ij}`. Further, let `b_i` be 1 if `i` is the source (starting) node; -1 if it is the sink (ending) node; and 0 otherwise. Then, optimal solution to the following program provides the shortest path from source to sink where `x_{ij}` will be 1 if `(i,j)` is on the shortest path. If there exists no path from source to sink, the problem will be infeasible.
`\min` `\sum_{j\in N} \text{ } \sum_{k \in N:(j,k)\in A} w_{jk} \times x_{jk}` (1)
`\text{subject to}` `\sum_{k \in N:(j,k)\in A}x_{jk} - \sum_{i \in N:(i,j)\in A}x_{ij} = b_{j}` `\forall j \in N` (2)
`x_{ij} >= 0` `\forall (i,j)\in A` (3)
