      VarD2 Class

      A 2-dimensional variable symbol, having two indices.

      Dimension Dimension of the variable symbol.
      Item Returns the variable at the (i,j)-th position. Note that the indices can be any scalar (constant for the mathematical model) expression.

      Consider for instance

      C#
      VarD2 x = Variable("x").Represents("flow on arc (i, j)").HasIndices(i, j).IsContinuous().IsNonnegative();
      Then:
      • x[0, 1] is the x-variable for arc (0, 1),
      • x[i, j] is the x-variable for arc (i, j),
      • x[0, j] is the x-variable for arc (0, j),
      • x[source[c], j] is the x-variable for arc (source[c], j), where source[c] is the source node of commodity c.
      Key Key of the variable.
      VariableType Type of the variable: Continuous, Binary or Integer.
