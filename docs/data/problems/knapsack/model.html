KP - Knapsack Problem

Definitions

Sets
`i`
available items
[0, 5]
Parameters
`v_{i}`
value of packing item i
values
`C`
total capacity of the knapsack
capacity
`w_{i}`
weight of item i
weights
Variables
`x_{i}`
1 if item i is packed in the knapsack; 0 otherwise
` \in { 0, 1 } `

Formulation

`\max` `\sum_{i}v_{i}x_{i}` (1)
`\text{subject to}` ` \sum_{i}w_{i}x_{i} \leq C` (2)
  1. knapsack_value: maximize total value of packed items in the knapsack
  2. knapsack_capacity: total weight of packed items cannot exceed the knapsack capacity
